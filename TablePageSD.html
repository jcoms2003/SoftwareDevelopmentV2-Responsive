Job Title Job Description Average Salary (beginner) Requirments
Front-end Developer Works on front end software. Uses markup languages such as HTML $60,060 to $72,102 per year Some experience in the following categories: Experience in markup languages (HTML), JavaScript, CSS, jQuery; Familiarity with browser testing and debugging; In-depth understanding of web development process; understanding of layout aesthetics; knowledge of SEO principles; knowledge of software like adobe script and photoshop; ability to work in a fast-paced environment; excellent analytical and multitasking skills; and a BS in a computer related field.
Back-end Developer Works on back-end development, uses programming languages such as: Python, Java, and C++ $95,000 to $120,000 annually You will need experience in the following areas to be a back-end developer: Proven work experience as a back-end developer; hands on practice with java, PHP, python, and other programming languages; basic familiarity with front-end languages; knowledge of CMS frameworks; and BS in computer science or other related fields.
Full stack Developer Works both front and back end, capable of developing websites independently $60,000 to $117,880 a year A full stack developer is responsible for the full cycle of an application. This means that you have the responsibility to create, design, and maintain a web application on both sides. You will use markup languages to create and design. Other programming languages are also used to develop and maintain the back end. So, in essence, a full stack developer is both a front-end and a back-end developer combined.
Mobile Developer Works on building mobile applications $73,000 to $151,000 a year A mobile developer will write code for iOS, Android, and Windows. You will need to understand mobile software systems and understand mobile application cycles. Programming languages, such as Java, Swift, and Objective-C are requirements for this job. They will also need to know web application development and cross-platform mobile kit.
Cloud and Server Developer Works with infrastructure (network, storage, servers, virtualization, and OS) and application issues $81,000 to $106,000 a year A cloud/server developer must create cloud/server-based solutions and applications. Knowledge of cloud computing systems like Amazon, Azure, or Google cloud. Server related skills are SQL and NoSQL. Java, HTML, and other programming languages are also essential for this job. Also knowledge of interchange formats, such as JSON, XL.
